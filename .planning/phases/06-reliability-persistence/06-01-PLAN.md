<name>Implement persistenceService.js</name>
<files>

- `tests/persistenceService.test.js`
</files>
<action>
1. Create `persistenceService.js` using the Revealing Module Pattern.
2. Initialize IndexedDB `pdf_unlocker_db` (v1).
3. Define `jobs` store: `id` (auto-increment), `status` (pending, processing, completed, failed), `timestamp`, `totalFiles`, `processedCount`.
4. Define `files` store: `id` (auto-increment), `jobId` (index), `name`, `status`, `originalBlob`, `outputBlob`, `hash`.
5. Implement CRUD operations: `createJob`, `updateJob`, `addFile`, `updateFile`, `getIncompleteJobs`.
</action>

<name>Integrate Persistence into WorkerPool</name>
<files>

</files>
<action>
